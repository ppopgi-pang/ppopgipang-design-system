# ppopgipang Design System

뽑기팡 서비스를 위한 디자인 시스템입니다.
컴포넌트, 아이콘, 토큰을 일관되게 사용할 수 있도록 정의합니다.

## Components

### Button

텍스트 기반 버튼 컴포넌트입니다.

| Prop              | Type                             | Default    | Description                |
| ----------------- | -------------------------------- | ---------- | -------------------------- |
| `label`           | `string`                         | —          | 버튼 텍스트 (필수)         |
| `primary`         | `boolean`                        | `false`    | primary / secondary 스타일 |
| `size`            | `'small' \| 'medium' \| 'large'` | `'medium'` | 버튼 크기                  |
| `backgroundColor` | `string`                         | —          | 배경색 오버라이드          |
| `onClick`         | `() => void`                     | —          | 클릭 핸들러                |

```tsx
import { Button } from '../components/button/Button';

<Button label="검색" primary size="medium" />;
```

---

### IconButton

아이콘만 표시하는 버튼 컴포넌트입니다. Tailwind 기반 크기 토큰을 사용합니다.

| Prop   | Type                                   | Default | Description            |
| ------ | -------------------------------------- | ------- | ---------------------- |
| `icon` | `ReactNode`                            | —       | 렌더링할 아이콘 (필수) |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'xl'`  | 버튼 크기              |

```tsx
import IconButton from '../components/button/IconButton';
import { ReactComponent as SearchIcon } from '../assets/icons/ic-search.svg';

<IconButton icon={<SearchIcon />} size="lg" />;
```

## Tokens

### iconSize

`src/tokens/iconSize.ts` 에 정의된 아이콘 크기 토큰입니다.

| Token | Tailwind Class      | Size |
| ----- | ------------------- | ---- |
| `xs`  | `w-3 h-3`           | 12px |
| `sm`  | `w-4 h-4`           | 16px |
| `md`  | `w-[18px] h-[18px]` | 18px |
| `lg`  | `w-5 h-5`           | 20px |
| `xl`  | `w-6 h-6`           | 24px |

```ts
import { iconSize, type IconSize } from '../tokens/iconSize';

// 사용 예시
const className = iconSize['lg']; // 'w-5 h-5'
```
